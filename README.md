## Movement Speeds
Phasmophobia uses a very easy understand movement system. Below you will see the list of movement speeds per entity.

- Ghost Hunting: 1.5
- Revenant Hunting (No Target): 1.1
- Revenant Hunting (Targeted): 3.0
